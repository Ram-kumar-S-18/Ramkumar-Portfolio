# Ramkumar S - Portfolio

A retro-modern, pixel-perfect portfolio website featuring a cyberpunk aesthetic, integrated AI assistant response themes, and multilingual support.

## Features
- **Retro-Modern Design**: High-contrast pixel art style with smooth animations and starfield backgrounds.
- **Multilingual Support**: Supports English, French, Spanish, German, Hindi, and Japanese.
- **Interactive Audio**: Custom hover sound effects and sound toggle (muted by default).
- **Mobile Responsive**: Custom hamburger menu and fluid layouts for all devices.
- **Tech Stack Icons**: Integrated Devicon symbols for a professional developer profile.
- **Utility Dock**: Quick links to Resume (Download), LinkedIn, WhatsApp, and Email.

## Tech Stack
- HTML5 (Semantic Structure)
- Vanilla CSS3 (Custom Variables & Modern Layouts)
- JavaScript (Client-side Logic & Canvas Animations)

## Deployment
This repository contains the production-ready source code. Simply host the `index.html` on any static hosting service (GitHub Pages, Netlify, Vercel).
